<h2>ScoreNotifier</h2>

A python program that I learnt from internet as part of understanding python to get the live score of a cricket match 
from cricinfo.com and display as notification on regular intervals.

<b>Requirements:</b>
<ul>
   <li>Pyton 2.x</li>
   <li>BeautifulSoup</li>
   <li>requests</li>
   <li>os</li>
   <li>sys</li>
   <li>time</li>
   <li>subprocess</li>
   </ul>
